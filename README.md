# AI Seller Agent

AI-powered Etsy listing optimization tool.

## Quick Start

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Requirements

- Node.js 18+
- MongoDB (optional for testing)

## Setup

1. Clone repository
2. Install dependencies (see above)
3. Configure backend/.env
4. Run both servers
